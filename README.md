# Multipage

## Description

This project was to get a deeper understanding on React Hooks. Here we created a web app with multiple pages (welcome and product pages). 

## Prerequisites

This was built with the help of:

* [React](https://reactjs.org/docs/getting-started.html)

## Installation

  1. System requirements:
      * [Node.js 12.22.0](https://nodejs.org/en/download/) or later
      * MacOS, Windows (including WSL), and Linux are supported.
  
  2. Clone the repository.
  3. ``npm install`` 
  4. ``npm start`` to start the server .
  5. open web page and go to url: http://localhost:3000
